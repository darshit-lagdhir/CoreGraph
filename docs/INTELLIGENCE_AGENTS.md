# CoreGraph: Intelligence Agents (Simplified)

This document explains how the system "thinks" about the data.

## 1. The Agential Cortex
We have an automated brain called the Agential Cortex. Instead of a human checking millions of nodes, this brain looks for patterns automatically.

## 2. Finding Problems
The agents constantly scan the data looking for anomalies (things that look wrong or dangerous). 
- If they find a risk, they highlight it in red on the visual map.
- They group problems together to show the "blast radius" (how far a problem spreads).

## 3. Safe Execution
These agents run in the background. Even if they are doing very heavy thinking, they never slow down the main visual map.
