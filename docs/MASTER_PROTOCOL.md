# CoreGraph: The Master Protocol (Simplified)

This document explains the overarching rules of the system.

## 1. Absolute Sovereignty
The system must always be in control. If a piece of data is missing, or a part of the system crashes, it must recover itself. Human intervention should not be required.

## 2. Speed over Detail
If the system is overloaded, it prioritizes keeping the visual screen alive (144Hz) over showing every single minor detail. It will catch up when it is safe.

## 3. Cryptographic Sealing
When the system finishes an analysis, it seals the result with a special code (SHA-384). This acts like a digital fingerprint. It proves that the result was generated by the machine and not faked by a human.
