# CoreGraph: Graph Analytics (Simplified)

This document explains how we understand the connections between our 3.81 million data points.

## 1. What is a Graph?
A graph is just a collection of dots (nodes) connected by lines (edges). We use this to see how different pieces of information relate to each other.

## 2. Finding Important Connections
We use math to find the most important dots in the graph. 
- If a dot has many lines connecting to it, it is a central hub.
- We group similar dots together into "clusters" so we can analyze them quickly.

## 3. Real-Time Math
Our engine calculates these connections instantly. It is so fast that even if you change the data, the math updates immediately without slowing down the visual map.
