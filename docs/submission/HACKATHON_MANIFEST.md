# COREGRAPH: HACKATHON SUBMISSION MANIFEST

## 1. WHAT DOES IT DO? (PROJECT OVERVIEW)
CoreGraph is a planetary-scale, Terminal-First OSINT (Open Source Intelligence) engine. Operating under a strict 150MB residency mandate, it dynamically resolves 3.81 million virtual nodes to identify supply chain vulnerabilities. It utilizes a Headless Forensic paradigm, stripping away WebGL bottlenecks to provide sub-atomic precision directly to the analyst terminal.

## 2. HOW WAS IT BUILT? (TECHNICAL STACK)
- **Intelligence Layer**: Google Gemini 1.5 Flash SDK orchestrating generative telemetry.
- **Router/API**: Asynchronous FastAPI bound with a custom Zero-Trust Security Middleware.
- **Persistence**: Flyweight-patterned In-Memory Vault at O(1) hash complexity.
- **Infrastructure**: Zero-CC Distroless Docker deployment targeted for edge nodes.

## 3. WHAT CHALLENGES DID YOU FACE?
The primary challenge was managing the 'Scale-Stability Paradox'—processing 3.81 million relational edges without breaching a 150MB heap limit. WebGL interfaces immediately crashed under load. We overcame this executing a 'Sovereign Headless Pivot', routing analytics through a background async pool and rendering metrics via a 144Hz CLI HUD.

## 4. WHAT IS THE INNOVATION?
We discarded traditional UI latency. CoreGraph introduces the 'Hadronic Binding Protocol'—calculating centrality in O(V+E) decoupled shards, ensuring bit-perfect execution with zero memory bloat. A live metric-aware interface allows for real-time forensic sovereignty without browser starvation.
